# 更新日志

所有版本的显著变更都将记录在此文件中。

本项目遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-01

### 新增

- 初始版本发布
- 添加 `Parser` 类，支持 URL 解析
- 添加 `parseMac()` 方法解析完整 URL
- 添加 `parseDomain()` 方法解析域名
- 添加 `parseController()` 方法解析控制器路径
- 添加示例文件和单元测试
